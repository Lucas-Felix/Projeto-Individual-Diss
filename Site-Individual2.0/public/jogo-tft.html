O que é TFT? O TFT é um jogo de batalhas automáticas, por isso é um representando de game “Auto Chess”. O nome vem de “Dota Auto Chess”, que é uma modificação do jogo Dota 2, que tempos depois virou um game próprio, o Dota Underlords, no mesmo estilo.
